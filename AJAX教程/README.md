## AJAX简介

AJAX是一种无需重新加载整个网页的情况下，能够更新部分网页的技术。

### 什么是AJAX？

AJAX=异步JavaScript和XML。

AJAX是一种用于创建快速动态网页的技术。

通过在后台与服务器进行少量数据交换，AJAX可以使网页实现异步更新。这意味着可以在不重新加载整个网页情况下，对网页的某部分进行更新。

传统的网页(不使用AJAX)如果需要更新内容，必需重载整个网页面。
